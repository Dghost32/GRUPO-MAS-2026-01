# Taller 1 - Patrones de Diseño

## Ejercicios

### Builder (`builder/`)

Patrón Builder para construir objetos `Automovil` paso a paso.

```bash
bun run dev:builder
```

### Bridge (`bridge/`)

Patrón Bridge para desacoplar notificaciones de plataformas (Desktop, Mobile, Web).

```bash
bun run dev:bridge
```

### Mediator (`mediator/`)

Patrón Mediator para gestionar la comunicación entre usuarios en un chat room.

```bash
bun run dev:mediator
```
