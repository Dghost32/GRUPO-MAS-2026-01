# GRUPO-MAS-2026-01

TypeScript project using Bun runtime for design pattern implementations.

## Requirements

- [Bun](https://bun.sh) >= 1.0

Install Bun:
```bash
curl -fsSL https://bun.sh/install | bash
```

## Setup

```bash
bun install
```

## Development

```bash
bun dev          # Run the project
bun test         # Run all tests
bun lint         # Lint with ESLint
```
